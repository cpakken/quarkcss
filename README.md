# QuarkCSS

Typed atomic CSS class composition for framework-agnostic utilities and styled components.

## Packages

- [@quarkcss/core](packages/core) - typed class name generators with variants, compounds, defaults, and composition.
- [@quarkcss/react](packages/react) - React styled components built on QuarkCSS core.
- [@quarkcss/solid](packages/solid) - Solid styled components built on QuarkCSS core.
- [@quarkcss/van](packages/van) - VanJS styled elements built on QuarkCSS core.

Use the package README for install instructions, examples, TypeScript helpers, and Tailwind conflict handling.
